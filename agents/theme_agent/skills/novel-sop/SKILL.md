# 小说创作SOP

流程：
1. 构思阶段
   1. 主题确定：明确小说的核心主题和主旨。主题关键词由用户填写
   2. 世界观设计：构建小说的世界观，例如物理规则、社会结构等，这个
   3. 人物设定：创建主要人物和配角，定义他们的背景、性格和动机。
2. 大纲阶段
   1. 整体大纲：设计小说的整体结构，包括开端、发展、高潮和结局。
   2. 章纲：将故事结构细化为章节，确定每章的主要内容和事件。
3. 写作阶段，每一章需要有下面的流程
   1. 初稿撰写：根据大纲撰写小说的初稿，注重故事流畅和情节发展。
   2. 修改润色：对初稿进行修改和润色，提升语言表达和故事质量。
   3. 审查反馈：审查小说内容在世界观和大纲是是否符合，并根据反馈进行调整。
      4. 
4. 发布阶段
   1. 格式调整：根据发布平台的要求调整小说格式。
