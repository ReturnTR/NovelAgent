# 小说创作SOP

整体流程如下：
1. 构思阶段
   1. 主题确定：明确小说的核心主题和主旨。主题关键词由用户填写
   2. 世界观设计：基于主题构建小说的世界观
   3. 人物设定：创建主要人物和配角
2. 大纲撰写阶段
   1. 整体大纲：设计小说的整体结构
   2. 卷纲：将故事结构细化为多个故事块
   3. 章纲：将故事块细分成每个章的内容，涉及到表要给表地址给到章纲
3. 内容写作阶段：
   1. 初稿撰写：根据大纲撰写小说的初稿，注重故事流畅和情节发展。
   2. 修改润色：对初稿进行修改和润色，提升语言表达和故事质量。

需要注意的是，小说不是一蹴而就的，需要渐进式，即一章一章的去写，具体表现为：
1. 先写最新一章的章纲
2. 基于章纲生成这一章的内容，包括初稿撰写，修改润色等
3. 基于内容，反过来对上面所有的内容，包括世界观、人物设定，以及之前的章节，大纲，卷纲，进行修补与完善
4. 进入下一章的写作，重复上述过程

同时，在进行每一部分的内容创作中（各个阶段的各个步骤），都需要对创作的结果进行审查，检查是否符合要求，如果不符合要求，需要进行修改，直到符合要求为止
