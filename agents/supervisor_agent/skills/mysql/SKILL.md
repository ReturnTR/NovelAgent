# Remote MySQL Executor Skill

## 技能描述
该技能用于执行远程MySQL数据库命令，支持查询和修改操作，返回执行结果。

## 功能说明
- 连接远程MySQL数据库服务器
- 执行SQL查询命令并返回结果集
- 执行SQL修改命令并提交事务
- 处理连接错误和异常情况
- 自动关闭数据库连接

## 使用方法
### 命令行使用实例
```bash
python remote_mysql_executor.py "SELECT * FROM table_name"
```

**输出举例**：
```
成功连接到MySQL服务器: 192.168.31.5:3306
[(1, '勇气', '面对困难和挑战的勇敢精神'), (2, '智慧', '知识和判断力的结合'), (3, '仁爱', '对他人的关爱和同情'), (4, '正义', '公平和道德的行为'), (5, '诚信', '诚实和守信用的品质')]
数据库连接已关闭
```

### 示例2：插入数据
```bash
python remote_mysql_executor.py "INSERT INTO humanity (name, description) VALUES ('宽容', '对他人错误的包容和谅解')"
```

**输出实例**：
```
成功连接到MySQL服务器: 192.168.31.5:3306
命令执行成功，影响行数: 1
数据库连接已关闭
None
```

## 注意事项
1. 确保MySQL服务器允许远程连接
2. 确保提供的用户名和密码具有相应的权限
3. 对于生产环境，建议使用环境变量或配置文件存储敏感信息，避免硬编码密码
4. 执行修改操作时要谨慎，特别是DELETE和UPDATE命令
5. 对于大型结果集，可能会占用较多内存

## 错误处理
- 连接错误：会打印错误信息并返回None
- SQL语法错误：会打印错误信息并返回None
- 权限错误：会打印错误信息并返回None

## 安全建议
1. 不要在脚本中硬编码密码，建议使用环境变量或加密的配置文件
2. 限制数据库用户的权限，遵循最小权限原则
3. 对输入的SQL命令进行验证，防止SQL注入攻击
4. 在生产环境中使用SSL连接加密数据库通信
