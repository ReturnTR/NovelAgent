# 总纲构建指南

## 一、总纲的核心要素

总纲是对整部小说“从哪开始、到哪结束、中间经历什么”的顶层设计。一个完整的总纲应包含以下五个要素：

| 要素 | 说明 | 
|------|------|
| **起点状态** | 故事开始时主角的处境、能力、关系、内心矛盾 |
| **终点状态** | 故事结束时世界的变化、主角的成长、最终选择 |
| **核心主线** | 主角从起点到终点必须经历的主要事件序列 |
| **阶段划分** | 将主线切分为3-5个大的阶段（即“卷”），每个阶段有独立目标 |
| **关键转折点** | 每个阶段内部推动剧情急转的重大事件 |

## 二、构建总纲的七步法

### 第1步：确定故事的“起点”与“终点”

**起点设计原则**：
- 主角必须有“缺失”或“困境”，让读者产生共情
- 起点与终点之间要有足够的“成长跨度”
- 起点应埋下最终答案的种子

**终点设计原则**：
- 必须回答起点提出的核心问题
- 主角必须做出“终极选择”，体现主题
- 世界状态发生不可逆改变

### 第2步：提炼核心主线（一句话 + 一段话）

**一句话主线**：
例如： 一个被判定为E级的废柴，用“解析”能力逆袭，最终揭开觉醒真相并决定人类进化方向。

**一段话主线（200字内）**：
例如： 2034年，全球觉醒十年后，林夜因技能无法直接攻击被GSMA判定为E级。他在镜城底层挣扎，意外发现自己能解析任何技能的弱点。在技能大赛中，他以E级身份连胜强敌，引起各方势力关注。拒绝被GSMA收编后，他被迫逃亡，在导师老姜的指导下掌握技能的本质，在自由城邦与技能贵族和隐修会的夹缝中寻找觉醒真相。最终他进入南极静默冰川，发现觉醒是地球维度升阶的预兆。面对四个终极选项，他拒绝被定义，创造出让人类自主选择是否进化的“第五种答案”。

### 第3步：划分阶段（卷）并设定每卷目标

根据主线，将故事切分为3-5卷，每卷有独立的目标、反派、成长阶段、爽点峰值。
